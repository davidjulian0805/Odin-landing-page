
# Odin Landing Page

This project is a **Landing Page** built as part of the **Foundations Course in The Odin Project**. The goal of this project is to practice structuring a webpage using **HTML** and styling it using **CSS with Flexbox**.

The design of the landing page was provided by The Odin Project, and the task was to recreate the layout as closely as possible.

---

## Project Overview

The landing page contains **four main sections and a footer**:

* Header / Navigation
* Hero Section
* Information Section
* Quote Section
* Call to Action Section
* Footer

The focus of this project was to:

* Practice **HTML page structure**
* Use **Flexbox for layout**
* Improve **CSS styling skills**
* Learn to organize a project step-by-step

---

## Project Instructions (From The Odin Project)

The main guidelines for the project were:

* Download the design images provided by The Odin Project.
* Recreate the landing page layout as closely as possible.
* Work **one section at a time**.
* Add all **HTML content first**, then apply CSS styling.
* Use **Flexbox** where necessary.
* Only one **CSS stylesheet** is required.
* The page **does not need to be responsive** for this project.
